# `catalog/` — Ken's reference catalog

The catalog is Ken's home for durable, browsable reference material, serving
four purposes:

- **Packages** — the standard-package sources (`catalog/packages/`), realized Ken
  code with proof/derivation manifests.
- **References** — catalog-wide citation conventions (`REFERENCES.md`), with
  per-entry references living in each entry.
- **Style** — a consistent entry format across every catalog area.
- **Discovery** — a browsable index so readers can find prior art by area
  without grepping the whole tree.

See `../docs/program/06-catalog-campaign.md` for the charter and
`../docs/program/07-catalog-style-guide.md` for the entry format.
