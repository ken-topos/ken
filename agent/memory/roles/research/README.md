# Research role memory scope

Lessons specific to the **Research** seat — the in-space investigation and
synthesis aux (see `agent/playbooks/federation/research.md`).

Read this scope plus `fleet/` and `enclave/` (Research sits on the enclave-facing
edge). Record a lesson here when it is Research-specific: a grounding trap in a
survey, a clean-room boundary call on a reference, a coordination-lane boundary
that had to be re-clarified. A genuinely cross-cutting lesson belongs at the
broadest scope where every reader must apply it, not here.

These are **lessons, not law** — verify a named file/flag/function still exists
before acting on one.
