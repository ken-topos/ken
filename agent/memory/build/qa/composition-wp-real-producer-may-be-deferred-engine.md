# Verify a composition WP's named producers are actually landed

For a **doc/composition WP** whose frame demands every AC "drive the **real**
producer" (the loudest producer-grep guardrail), do NOT trust the producer
*name* — grep that it's a **landed executable**, because a composition WP rides
the seam's downstream **discharge engines**, and those are commonly **deferred**
while only the **export PROJECTION** is landed. Two live catches on B4-agentic
(spec-author concurred + fixed both):

- **AC3 metamorphic** cited "the *real* `OQ-relational` 2-run reduction + L2
  test-gen" — but the product-program reducer is **`[rel-deferred]`** (`61 §5.3`
  "named not faked"; `ifc.rs:376` `[Sec1-reduce]`, `ifc.rs:357`
  `TRIGGER_REL_DEFERRED`, `sec1_acceptance.rs:313` `// Simulate: product(c,ζ)…`)
  and L2 test-gen has **zero** crate producer.
- **AC4 RV** cited "monitored **end-to-end** … violation **caught**" — but the
  live Büchi monitor / `compile` faithfulness is **`(oracle)`/B2-deferred**
  (`seed-trace` TR-E pins only the projection-as-projection).

**Fix (the static-vs-runtime-face split for composition WPs):** the **landed**
producer is the **status→field projection through the real export emitter**
(`export.rs`) — the same face the fully-landed ACs drive; the discharge engine
(reducer / monitor / test-gen) is a **named-deferred runtime face carried as a
trigger** (`[rel-deferred]`/`(oracle)`), NOT asserted "real." Precedent lives in
the seam already — `seed-ifc` D3/D4/G1 (by-proof verdict-mapping +
`[rel-deferred]`, driving no reducer) and `seed-trace` TR-E. So a composition
corpus is a **projection/partition-fidelity** corpus, not engine-execution.

**Tell:** an AC that names a producer the frame calls "real" but whose grep
lands on a `[rel-deferred]`/`(oracle)`/`// Simulate` marker — that's the
hand-feed trap wearing a real-producer label. Extends conformance hand feeds the
deliverable and soundness AC static vs runtime face; sibling of kernel backed
claim grep the emission not the name (verify the mechanism, not the name).
